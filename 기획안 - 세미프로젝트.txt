팀 명 : 잔디인형과 메밀차
팀 원 : 홍성빈, 강수정, 김대욱, 남지선, 김정민, 민경은

1. 프로젝트 제목 : FIVE STAR TICKET(부제 : 오성티켓)
2. 개요 : 뮤지컬, 연극, 전시 등 각종 문화생활을 즐길 수 있는 티켓 예매 사이트 제공
3. 구현 기능 
- 회원가입 및 로그인 서비스
- 마이페이지(포인트, 예약조회/취소, 회원정보수정, 회원탈퇴) 서비스
- 공연정보조회 서비스
- 공연상세 서비스(네비게이터, 리뷰, 지도 API) 
- 월간 랭킹서비스
- 예매/예약 서비스
- 고객센터(FAQ, 1:1문의) 서비스
- 관리자(회원관리, FAQ, 1:1문의, 공연등록) 서비스
4. 설계의 주안점
- 회원가입 : 네이버, 카카오 간편 로그인 API 서비스 제공 
- 검색 : 뮤지컬, 연극, 전시 통합검색 및 검색어 자동완성 기능 제공
- 디자인 : 사용자 입장에서 생각한 UI 구현
- 공연추천 : 관리자가 추천하고 싶은 공연 서비스 제공 
- 월간 랭킹 : 월간 및 카테고리별 공연랭킹조회 서비스 제공
- 공연상세 : 네비게이터, 지도 API, 리뷰등록 서비스 제공
- 공연예매 : 사용자의 편의를 고려한 화면 내 원클릭으로 공연일자 변경기능
                 첫화면으로 돌아가 정보를 재입력받을 필요없는 편리성
- 마이페이지 : 사용자 입장을 고려한 토탈 간편 서비스 제공
  	      최근 예매내역, 최근 포인트 적립 및 사용내역, 나의 리뷰 서비스 제공
5. 사용기술 및 개발환경
- 운영체제 : Window OS / MAC OS 
- 개발툴 : Oracle SQL Developer, Server : Apache Tomcat 8.5, Eclipse E, Visual Studio Code
- 사용기술 :  Java, JavaScript, JQuery, JQuery UI, HTML, CSS, JSP, GIT, Ojdbc, 지도 API, 간편로그인 API
- 디자인툴 : Kakao oven, ERD cloud, Diagrams
6. 팀원별 역할
- 홍성빈 : (공연팀)세미 프로젝트 총괄 / 공연예매 결제 서비스 및 디자인 구현   
- 강수정 : (공연팀)랭킹 / 공연상세페이지 / 리뷰 서비스 및 디자인 구현
- 김대욱 : (공연팀)DB 관리자 / 검색서비스 / 메인페이지 서비스 및 디자인 구현
- 남지선 : (멤버팀)로그인 및 회원가입 서비스 구현 / 고객센터, 포인트 서비스 및 디자인 구현 / 공연등록 서비스 및 디자인 구현
- 김정민 : (멤버팀)GIT 관리자 / 회원정보 및 탈퇴, 예매내역 및 디자인 구현 / 공연등록 서비스 및 디자인 구현 
- 민경은 : (멤버팀)로그인 및 회원가입 디자인 구현 / 고객센터 및 1:1문의 답변 / 나의 리뷰 서비스 및 디자인 구현
